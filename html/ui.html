Starter Checklist: This will help you get started on the server for the first time!

Use "ESC > Settings > Voice Chat" to set up audio / voice settings.

Set up keybinds in "ESC > Settings > Key Bindings > FiveM (Scroll down!)

Set up graphics and display options in "ESC > Settings"

Set up your character clothing & face options in the room.

Take note of your character's ID number (Do /id in chat)

Familiarize yourself with keybinds and what they all do.

Enjoy the server and thanks for playing!
